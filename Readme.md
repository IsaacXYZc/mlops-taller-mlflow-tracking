# Taller MLflow Tracking

# Instrucciones
1. Iniciar ollama con `ollama start`.
2. Instalar el entorno virtual de Python `venv\Scripts\activate`.
3. Instalar las dependencias con `pip install -r requirements.txt`.
4. Abrir la interfaz de MLflow con `mlflow ui`.
5. Ejecutar el notebook mlflow-tracking.ipynb.
